04 / TAG MANAGEMENT

Labels that Bring Order ស្លាកដែលនាំមកនូវរបៀប

Tags are the organisational backbone across the platform. Create colour-coded labels, assign them to users and conversations, and filter everything by them — instantly. ស្លាកគឺជាស្នូលនៃការរៀបចំនៃវេទិកា។ បង្កើតស្លាកពណ៌ ដាក់ចំពោះអ្នកប្រើ និងការសន្ទនា ហើយត្រងរាល់អ្វីដោយភ្លាម។

  • Visual tag library as coloured pills with usage countsឃ្លាំងស្លាកបង្ហាញជាពណ៌ ជាមួយចំនួនការប្រើ
  • Create with name + colour picker (preset or custom)បង្កើតដោយឈ្មោះ + ជ្រើសពណ៌ (ស្ដង់ដារ ឬផ្ទាល់ខ្លួន)
  • Edit or delete tags — cascading removal from all recordsកែ ឬលុបស្លាក — លុបពីកំណត់ត្រាទាំងអស់ស្វ័យប្រវត្តិ
  • Assign to users via editor, or to chats via detail modalដាក់ស្លាកចំពោះអ្នកប្រើ ឬការសន្ទនា
